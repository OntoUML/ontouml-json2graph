json2graph.lib_json2graph

It provides a convenient interface for converting OntoUML JSON files into knowledge graphs, with the flexibility to customize the output and control the execution mode for different use cases.

json2graph.lib_json2graph.decode_ontouml_json2graph(json_path, graph_format='ttl', language='', execution_mode='production')

Main function for ontouml-json2graph.

Parameters:

  • json_path (str) – Path to the JSON file to be decoded provided by the user.

  • graph_format (str) – Format for saving the resulting knowledge graph. Default value is ‘ttl’ (Turtle syntax).

  • language (str) – Language tag to be added to the ontology’s concepts.

  • execution_mode (str) – Information about execution mode. Valid values are ‘production’ (default) and ‘test’.

Returns:

Saved output file path. Used for testing.

Return type:

str
